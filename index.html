Android UML Generator

Generate a sequence diagram based upon the usage of an Android Application Members: Miles Peele, Tristan Vernon, Jake Shor, Tim Fulton

View project on GitHub

Purpose

Execution flows of applications can sometimes be difficult to visualize without having UML diagrams. In particular, sequence diagrams allow for the capture of flow logic, logic methods, usage program, and logic of services of a program.

The goal of this project is to automatically generate sequence diagrams based on the usage of an application. We also generate an XMI file so that the UML diagram can be imported into StarUML.
